# Wordly – Single Page Application (SPA)

Wordly is a simple Single Page Application (SPA) that allows users to search for words and view their meanings in real time. The application fetches data from a public dictionary API and displays results dynamically without reloading the page.

---

## Features
- Search for a word using a search form
- Display word definitions, part of speech, and examples
- Show pronunciation text and audio (when available)
- Display synonyms for searched words
- Handle errors for invalid or empty searches
- Dynamic content updates using JavaScript (SPA behavior)

---

## API Used
- **Free Dictionary API**  
  https://dictionaryapi.dev/

---

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API

---

## How to Run the Project
1. Clone or download this repository
2. Open the project folder in VS Code
3. Open `spa.html` in a web browser  
 

---

## Usage Instructions
1. Type a word into the search input field
2. Click the **Search** button
3. View the word’s definition, pronunciation, and related information
4. Click **Clear** to reset the search results
